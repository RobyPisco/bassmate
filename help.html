🎸 Studio — Il Manico Interattivo

Il cuore di Bassmate. Visualizza scale, accordi, arpeggi e modi su un manico completamente interattivo.

  • Nota radice: clicca uno dei pulsanti C D E… o usa le frecce ◀ ▶ per trasporre di un semitono.
  • Scala / Accordo: naviga fra i tab Base, Modi, Arpeggi, Altro per selezionare la formula. Il manico si aggiorna istantaneamente.
  • Numero di corde: scegli tra 4, 5 o 6 corde dal selettore in cima.
  • Accordatura: Standard, Drop D, Eb, Drop Db, DADGAD e altre varianti disponibili nel selettore.
  • Etichette: scegli cosa mostrare su ogni nota — Gradi (1 b3 5…), Note (C D E…), Solfeggio (Do Re Mi…) o Dita per i box.
  • Mancino: attiva la modalità specchio per chi suona con la mano destra sul manico.
  • Visualizzazione: selettore "Tutte / Solo zona" per filtrare le note per posizione sul manico.
  • Suono: clicca qualsiasi nota sul manico per sentirla riprodotta dal sintetizzatore interno.
  • Reset Fretboard: pulsante rapido per azzerare il manico senza toccare le impostazioni.

🎯 Quiz Mode

Metti alla prova la memoria delle note sul manico. Tre modalità di difficoltà, timer configurabile, sistema di combo e statistiche persistenti.

  • Avvio: clicca ▶ Start nel pannello Quiz. Un countdown di 3 secondi precede la prima domanda.
  • Domanda: una nota sul manico si illumina — indovina usando i pulsanti in basso (C, D, E… o Do, Re, Mi…).
  • Modalità: Note (indovina il nome della nota) o Grado (indovina l'intervallo).
  • Difficoltà: Facile, Normale, Difficile — cambia la velocità e la zona del manico coinvolta.
  • Timer: imposta 30s, 60s, 90s o 120s. Alla scadenza vedi il punteggio finale.
  • Combo: risposte consecutive corrette moltiplicano il punteggio. Errore = combo reset.
  • Statistiche: punteggio record, sessioni totali, nota più debole — salvati localmente.

🎙️ Accordatore Cromatico

Accordatore professionale in tempo reale tramite microfono.

  • Clicca il tab Tuner nella barra di navigazione.
  • Premi Avvia Tuner e concedi il permesso al microfono quando richiesto.
  • Avvicina il basso al microfono e suona una corda aperta.
  • La lancetta indica se la nota è calante (flat) o crescente (sharp). Diventa verde quando sei perfettamente intonato.
  • Mostra la nota rilevata, la frequenza in Hz e i cents di scostamento.

🥁 Metronomo

Metronomo integrato per studiare le scale a tempo preciso.

  • BPM: regola da 30 a 240 BPM con slider o input diretto.
  • Battuta: scegli tra 2/4, 3/4, 4/4, 6/8.
  • Accento: il primo beat è evidenziato in oro per trovare facilmente il "battere".
  • Flash: attiva il flash visivo sincronizzato con il beat per chi suona in luoghi rumorosi.
  • Timer sessione: imposta un countdown per la sessione di studio (es. 10 minuti).
  • Suono: puoi continuare a cliccare le note sul manico mentre il metronomo batte.

📝 Trascrizione Tab (ASCII)

Registra le tue idee musicali mentre suoni — genera tab ASCII in tempo reale.

  • Attiva il tab Transcription dalla barra di navigazione.
  • Clicca le note sul manico: vengono aggiunte automaticamente alla tablatura.
  • Usa Copia Tab per copiare il risultato e incollarlo ovunque (chat, email, documenti).
  • Usa Cancella per svuotare la trascrizione e ricominciare.

📋 Manici Vuoti da Stampare

Strumento per insegnanti e studenti: genera fogli di lavoro con manici da compilare.

  • Clicca l'icona 📋 nell'header per aprire la pagina Grids.
  • Corde: scegli 4, 5 o 6 corde.
  • Numero griglie: da 1 a 12 manici per foglio.
  • Titolo per griglia: ogni manico ha un campo titolo editabile (es. "Scala di Do maggiore").
  • Posizione fret: imposta da quale tasto parte ogni griglia (es. "da fret 5").
  • Esporta Scala: scegli tonica + tipo di scala e applica le note direttamente su una griglia o su tutte. Le note vengono disegnate come cerchi colorati con il nome (viola = tonica).
  • Scale disponibili: Maggiore, Minore Naturale, Pentatonica Maj/Min, Blues, Dorian, Mixolydian, Minore Armonica, Minore Melodica.
  • Clicca 🖨️ Stampa Ora — i controlli spariscono e rimane solo il foglio pulito.

📚 Esercizi Pratici

Raccolta di esercizi guidati per lo studio del basso.

  • Clicca l'icona 📚 nell'header per aprire la pagina Esercizi.
  • Sfoglia gli esercizi divisi per categoria e difficoltà.
  • Ogni esercizio include diagramma, descrizione e suggerimenti pratici.

📸 Snapshot & 🖨️ Stampa

  • 📸 Snapshot: esporta il manico corrente come immagine PNG ad alta risoluzione — ideale per social, messaggi o appunti digitali.
  • 🖨️ Stampa PDF: genera una stampa professionale del manico in formato A4 orizzontale, ottimizzata per la carta.

❤️ Preferiti & Condivisione

  • Salva preferito: clicca ❤️ per salvare il setup corrente (nota radice, scala, accordatura). Puoi salvarne quanti vuoi.
  • Richiama preferito: usa il menù preferiti (icona segnalibro) per caricare un preset salvato in un click.
  • Elimina preferito: seleziona un preferito e clicca l'icona cestino 🗑️ che appare nell'header.
  • Condividi link: clicca 🔗 — viene copiato negli appunti un URL con tutta la configurazione corrente (radice, scala, accordatura, etichette, mano). Incollalo e condividilo: chiunque aprirà esattamente la stessa visualizzazione.

⚙️ Impostazioni

  • Tema 🌙 / ☀️: passa da Dark a Light mode. La preferenza viene salvata automaticamente.
  • Audio 🔊 / 🔇: attiva o disattiva la riproduzione sonora delle note.
  • Lingua: pulsanti IT / EN nell'header per cambiare lingua in tempo reale.
  • Mancino: selettore "Right / Left" nel pannello Studio — specchia l'intero manico.
  • Accordatura: Standard 4/5/6 corde, Drop D, Eb, Drop Db, DADGAD.

⌨️ Scorciatoie Tastiera (Desktop)

  • ← / → — cambia nota radice (semitono su/giù)
  • Q — avvia / ferma Quiz
  • M — apri / chiudi Metronomo
  • R — reset fretboard
  • ? — mostra pannello scorciatoie

🎸 Studio — Interactive Fretboard

The core of Bassmate. Visualize scales, chords, arpeggios and modes on a fully interactive fretboard.

  • Root note: click any of the C D E… buttons or use the ◀ ▶ arrows to transpose by semitone.
  • Scale / Chord: navigate the tabs Basic, Modes, Arpeggios, Other to select a formula. The fretboard updates instantly.
  • String count: choose 4, 5 or 6 strings from the top selector.
  • Tuning: Standard, Drop D, Eb, Drop Db, DADGAD and more available from the tuning selector.
  • Labels: choose what to display on each note dot — Degrees (1 b3 5…), Notes (C D E…), Solfege (Do Re Mi…) or Fingers for box positions.
  • Lefty mode: mirror the entire fretboard for left-handed players.
  • View filter: "All / Zone only" selector to show notes only in a specific position.
  • Sound: click any dot on the fretboard to hear it played by the built-in synthesizer.
  • Reset Fretboard: quick button to clear the fretboard without changing settings.

🎯 Quiz Mode

Test your fretboard note memory. Three difficulty levels, configurable timer, combo multiplier system and persistent stats.

  • Start: click ▶ Start in the Quiz panel. A 3-second countdown precedes the first question.
  • Question: a note on the fretboard lights up — guess it using the buttons below (C, D, E… or Do, Re, Mi…).
  • Mode: Note (guess the note name) or Degree (guess the interval).
  • Difficulty: Easy, Normal, Hard — affects speed and the fretboard zone used.
  • Timer: set 30s, 60s, 90s or 120s. When it runs out, your final score is shown.
  • Combo: consecutive correct answers multiply your score. A mistake resets the combo.
  • Stats: high score, total sessions, weakest note — saved locally on your device.

🎙️ Chromatic Tuner

Professional real-time tuner via microphone.

  • Click the Tuner tab in the navigation bar.
  • Press Start Tuner and grant microphone permission when prompted.
  • Hold your bass near the microphone and pluck an open string.
  • The needle shows whether the note is flat or sharp. It turns green when perfectly in tune.
  • Displays detected note, frequency in Hz and cents deviation.

🥁 Metronome

Built-in metronome to practice scales in time.

  • BPM: set from 30 to 240 BPM with slider or direct input.
  • Time signature: choose between 2/4, 3/4, 4/4, 6/8.
  • Downbeat accent: beat 1 is highlighted in gold to help you find "the one".
  • Flash: enable a visual flash synced to the beat — useful in noisy environments.
  • Session timer: set a countdown for your practice session (e.g. 10 minutes).
  • Sound: you can continue clicking fretboard notes while the metronome ticks.

📝 Tab Transcription (ASCII)

Record your musical ideas while playing — generates ASCII tablature in real time.

  • Switch to the Transcription tab in the navigation bar.
  • Click notes on the fretboard: they are automatically added to the tab.
  • Use Copy Tab to copy the result and paste it anywhere (chat, email, documents).
  • Use Clear to erase the transcription and start fresh.

📋 Printable Fretboard Sheets

Tool for teachers and students: generate worksheets with blank or pre-filled fretboard diagrams.

  • Click the 📋 icon in the header to open the Grids page.
  • Strings: choose 4, 5 or 6 strings.
  • Grid count: from 1 to 12 fretboards per sheet.
  • Grid title: each fretboard has an editable title field (e.g. "C Major Scale").
  • Fret position: set which fret each grid starts from (e.g. "from fret 5").
  • Export Scale: choose a root + scale type and apply the notes to one grid or all. Notes are drawn as colored circles with name (purple = root).
  • Available scales: Major, Natural Minor, Major/Minor Pentatonic, Blues, Dorian, Mixolydian, Harmonic Minor, Melodic Minor.
  • Click 🖨️ Print Now — controls hide and only the clean sheet remains.

📚 Practice Exercises

Curated exercise library for bass study.

  • Click the 📚 icon in the header to open the Exercises page.
  • Browse exercises sorted by category and difficulty.
  • Each exercise includes a diagram, description and practical tips.

📸 Snapshot & 🖨️ Print

  • 📸 Snapshot: export the current fretboard as a high-resolution PNG — great for social media, messages or digital notes.
  • 🖨️ Print PDF: generate a professional A4 landscape print of the current fretboard, optimized for paper.

❤️ Favorites & Sharing

  • Save favorite: click ❤️ to save the current setup (root note, scale, tuning). Save as many as you like.
  • Load favorite: use the favorites menu (bookmark icon) to load a saved preset in one click.
  • Delete favorite: select a favorite and click the trash icon 🗑️ that appears in the header.
  • Share link: click 🔗 — a URL encoding your full configuration (root, scale, tuning, labels, hand) is copied to clipboard. Paste it and share: anyone who opens it sees exactly the same view.

⚙️ Settings

  • Theme 🌙 / ☀️: toggle Dark / Light mode. Your preference is saved automatically.
  • Audio 🔊 / 🔇: enable or disable note playback sound.
  • Language: IT / EN buttons in the header — switches the entire UI in real time.
  • Lefty: "Right / Left" selector in the Studio panel — mirrors the entire fretboard.
  • Tuning: Standard 4/5/6 string, Drop D, Eb, Drop Db, DADGAD.

⌨️ Keyboard Shortcuts (Desktop)

  • ← / → — change root note (semitone down/up)
  • Q — start / stop Quiz
  • M — open / close Metronome
  • R — reset fretboard
  • ? — show shortcuts panel
