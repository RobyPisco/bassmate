# 🎸 Bassmate
**The Ultimate Interactive Bass Fretboard Explorer**

Bassmate is a professional, lightweight, and offline-capable web application designed for bassists of all levels. Explore scales, arpeggios, and intervals across various tunings and string configurations (4, 5, and 6 strings).

## ✨ Features
- **Interactive Fretboard:** Visualize scales (Major, Minor, Modes, Blues, etc.) and arpeggios in real-time.
- **Multilingual Support:** Full support for Italian (Solfeggio: Do, Re, Mi) and English (Scientific: C, D, E) nomenclature.
- **Quiz Mode:** An interactive engine to test and improve your fretboard note recognition.
- **Favorites System:** Save your favorite scale/tuning combinations locally and access them instantly.
- **Teacher Mode (Blank Grids):** A dedicated tool for teachers to generate and print blank fretboard templates (4, 5, or 6 strings).
- **Practical Exercises:** Includes 10 foundational routines for technique and theory development.
- **Offline First (PWA):** Installable on mobile devices. Works perfectly in the rehearsal room without internet.
- **Professional Print Export:** Export the current fretboard view to a high-quality landscape A4 PDF.

## 🚀 Getting Started
Simply open `index.html` in your browser or host it on any static web server (like GitHub Pages).

## 🛠️ Built With
- **Vanilla JavaScript:** Zero dependencies for maximum speed and longevity.
- **Modern CSS:** Utilizing CSS variables and responsive design for a premium look.
- **SVG Engine:** Mathematical precision for fretboard rendering and printing.
- **Service Workers:** Ensuring offline reliability.
